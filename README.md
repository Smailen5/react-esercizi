## In locale

- una volta scaricato la cartella principale ricorda di usare `cd inserisci-nome-progetto` che vuoi visualizzare.
- `npm install`
- `npm run dev`
- apri il tuo browser al [link](http://localhost:5173/) per visualizzare il progetto in locale.


## Sviluppo

- Bootstrap
- Tailwind css
- React+vite
